# Rei

Freelance Tool Calculator with a landing page, calculator app, and blog/reference page.

## Pages

- `index.html` - Landing page
- `calculator.html` - Main Freelance Tool Calculator app
- `blog.html` - Blog and references page

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite (usually `http://localhost:5173`).
